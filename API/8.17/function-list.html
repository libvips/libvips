libvips has a set of operators, each of which computes some useful image processing operation. Each operator is implemented as a GObject class, for example VipsGamma. Classes are identified by their unique VipsObject:nickname, in this case gamma.

From the command-line, C++ and most language bindings, you use the nickname to call the operator. For example in C++:

vips::VImage fred = ...;
vips::VImage jim = fred.gamma();

or Python:

fred = jim.gamma()

libvips has a set of C wrapper functions for calling operators, in this case vips_gamma():

VipsImage *fred = ...;
VipsImage *jim;

if (vips_gamma(fred, &jim, NULL))
    ...error;

Some operators have many C convenience functions.
