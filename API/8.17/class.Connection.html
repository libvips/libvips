An abstract base class representing a source or sink of bytes.

It can be connected to a network socket, for example, or perhaps a Node.js stream, or to an area of memory. This allows it to support operations like JPEG loading, see for example vips_jpegload_source().

Subclass to add other input sources. Use VipsSourceCustom and VipsTargetCustom to make a source or target with action signals. These classes provide action signals such as:

  • VipsSourceCustom::read for reading data from a custom source.
  • VipsSourceCustom::seek for seeking within a data stream.
  • VipsTargetCustom::write for writing data to a custom target.
