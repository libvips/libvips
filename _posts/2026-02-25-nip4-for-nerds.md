title: nip4 for nerds
---

[nip4 is now mostly done](https://github.com/jcupitt/nip4), so I thought I'd
write a post on nip4's programming language.

nip4 has its own programming language, called `snip`. You can use it like any
programming language, for example:

```
#!/usr/bin/env snip

main = "hello, world!\n";
```
